🤝 Contribuir

Help expand and improve these open datasets

How to Contribute

1
Fork el repositorio
Ve a github.com/juanmoisesd y haz fork del repositorio de dataset relevante.
2
Añade o corrige datos
Edita el archivo CSV/Excel añadiendo datos según el formato de variables en CODEBOOK.md.
3
Envía un Pull Request
Envía un PR con una descripción clara de qué cambió y por qué.
4
Proceso de revisión
Todas las contribuciones se revisan en 7 días. Las contribuciones aceptadas se acreditan en CHANGELOG.md.
5
Alternativa: Abre un Issue
Si encuentras un error o quieres sugerir datos sin PR, abre un GitHub Issue.
